# FastAPI

- 下記コマンドでAPIの立ち上げ
```
docker-compose up
```

- 下記コマンドでアクセスする
```
http://localhost:8000/docs
```

- 下記コマンドでDBを確認することができる
```
docker-compose exec db mysql demo
```
